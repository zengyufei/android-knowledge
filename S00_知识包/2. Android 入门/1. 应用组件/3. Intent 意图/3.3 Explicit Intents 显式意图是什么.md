# 🎯 显式意图

🔍 显式意图主要在应用程序自身的边界内使用。在显式意图中，你需要指定响应该意图的组件。

## 🌟 特点

- 🎯 需要明确指定目标组件
- 💬 通常用于应用内部通信

## 🛠️ 指定目标组件的方法

显式意图必须通过以下方法之一来指定目标组件：

- 🧩 `setComponent(ComponentName)`
- 🏫 `setClass(Context, Class)`
- 📝 `setClassName(String, String)`

## 🚀 常见用途

1. 📱 启动活动
2. 📢 广播消息
3. 🔧 启动应用内的服务

## ⚙️ 处理机制

🔄 显式意图不由系统解析，而是直接传递给意图中指定的组件。
