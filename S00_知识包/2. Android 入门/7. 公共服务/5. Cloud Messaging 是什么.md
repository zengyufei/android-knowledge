# 🚀 Cloud Messaging

🔥 Firebase Cloud Messaging (FCM) 是一个强大的、节能的消息服务，使您能够可靠且安全地向您的 Android 应用程序发送消息。它允许您发送两种类型的消息：

1. 📢 "通知消息"：
   - 主要用于用户通知
   - 仅在应用程序处于前台时才会传递

2. 📊 "数据消息"：
   - 即使应用程序在后台或被终止也可以处理
   - 可用于发送自定义的键值对

🌟 FCM 还支持多种附加功能，例如：

- 📡 主题消息：向订阅同一主题的多个设备发送消息
- 👥 设备组消息：向用户设备组发送消息
- 🔄 上行消息：从客户端应用程序向 FCM 服务器发送消息
