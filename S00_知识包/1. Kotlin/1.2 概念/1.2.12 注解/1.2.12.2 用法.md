 
## 🚀 用法

```kotlin
@Fancy class Foo {
    @Fancy fun baz(@Fancy foo: Int): Int {
        return (@Fancy 1)
    }
}
```

🔧 如果需要对类的主构造函数进行标注，则需要在构造函数声明中添加 `constructor` 关键字。

#### 🏷️ 并将注解添加到其前面：

```kotlin
class Foo @Inject constructor(dependency: MyDependency) { …… }
```

#### 🔍 你也可以标注属性访问器：

```kotlin
class Foo {
    var x: MyDependency? = null
        @Inject set
}
```
