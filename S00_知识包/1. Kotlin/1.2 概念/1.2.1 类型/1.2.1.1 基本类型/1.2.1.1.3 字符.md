### 字符
#### 字符用 <code>Char</code> 类型表示。 字符字面值用单引号括起来: <code>'1'</code>。

> 在 JVM 上，字符作为原始类型 char 存储，表示一个 16 位的Unicode字符。

#### 特殊字符可以以转义反斜杠 \ 开始。 支持这几个转义序列：

- <code>\t</code>——制表符
- <code>\b</code>——退格符
- <code>\n</code>——换行（LF）
- <code>\r</code>——回车（CR）
- <code>\\'</code>——单引号
- <code>\\"</code>——双引号
- <code>\\\\</code>——反斜杠
- <code>\\$</code>——美元符

#### 编码其他字符要用 Unicode 转义序列语法：<code>'\uFF00'</code>。


```kotlin
fun main() {
//sampleStart
    val aChar: Char = 'a'

    println(aChar)
    println('\n') // 输出一个额外的换行符
    println('\uFF00')
//sampleEnd
}
```

如果字符变量的值是数字，那么可以使用 <code>digitToInt()</code> 函数将其显式转换为 Int 数字。

> 在 JVM 平台，当需要可空引用时字符会是装箱的 Java 类，类似于数字。 装箱操作不保留同一性。
