 
## 二元操作 🔢

### 算术运算符 🧮

| 表达式     | 翻译为        |
| -----------|-------------- |
| `a + b` | `a.plus(b)` |
| `a - b` | `a.minus(b)` |
| `a * b` | `a.times(b)` |
| `a / b` | `a.div(b)` |
| `a % b` | `a.rem(b)` |
| `a..b` | `a.rangeTo(b)` |
| `a..<b` | `a.rangeUntil(b)` |

🔍 对于此表中的操作，编译器只是解析成*翻译为*列中的表达式。

#### 🌟 下面是一个从给定值起始的 `Counter` 类的示例，它可以使用重载的 `+` 运算符来增加计数：

```kotlin
data class Counter(val dayIndex: Int) {
    operator fun plus(increment: Int): Counter {
        return Counter(dayIndex + increment)
    }
}
```
