 
### 🔄 返回 Unit 的函数

🚫 如果一个函数并不返回有用的值，其返回类型是 `Unit`。

🧩 `Unit` 是一种只有一个值——`Unit` 的类型。

#### 🔁 这个值不需要显式返回：

```kotlin
fun printHello(name: String?): Unit {
    if (name != null)
        println("Hello $name")
    else
        println("Hi there!")
    // `return Unit` 或者 `return` 是可选的
}
```

🌟 `Unit` 返回类型声明也是可选的。

#### 🔍 上面的代码等同于：

```kotlin
fun printHello(name: String?) { …… }
```
