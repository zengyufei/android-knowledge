## 🏷️ Break 与 Continue 标签

🔖 在 Kotlin 中任何表达式都可以用标签来标记。
🏷️ 标签的格式为标识符后跟 `@` 符号，例如：`abc@`、`fooBar@`。
➕ 要为一个表达式加标签，我们只要在其前加标签即可。

```kotlin
loop@ for (i in 1..100) {
    // ……
}
```

🔀 现在，我们可以用标签限定 `break` 或者 `continue`：

```kotlin
loop@ for (i in 1..100) {
    for (j in 1..100) {
        if (……) break@loop
    }
}
```

↩️ 标签限定的 `break` 跳转到刚好位于该标签指定的循环后面的执行点。
🔄 `continue` 继续标签指定的循环的下一次迭代。
