## 🌟 显式的 Job

🚀 `launch` 协程构建器返回一个 `Job` 对象，它是已启动协程的句柄，可用于显式等待其完成。

#### 🔍 例如，你可以等待子协程完成，然后打印 "Done" 字符串：

```kotlin
import kotlinx.coroutines.*

fun main() = runBlocking {
    val job = launch { // 启动一个新的协程并保持对其Job的引用
        delay(1000L)
        println("World!")
    }
    println("Hello")
    job.join() // 等待子协程完成
    println("Done")
}
```

#### 🎭 这段代码产生以下输出：

```text
Hello
World!
Done
```
