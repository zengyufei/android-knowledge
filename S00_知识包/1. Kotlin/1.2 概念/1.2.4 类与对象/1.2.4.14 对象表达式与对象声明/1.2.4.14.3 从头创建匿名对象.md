### 🎨 从头创建匿名对象

🚀 对象表达式以 `object` 关键字开始。

#### 🧩 如果你只需要一个没有任何非平凡超类型的对象，可以在 `object` 后的大括号中写入其成员：

```kotlin

fun main() {
//sampleStart
    val helloWorld = object {
        val hello = "Hello"
        val world = "World"
        // 对象表达式继承自Any，所以在`toString()`上需要使用`override` 🧬🔄
        override fun toString() = "$hello $world"
    }

    print(helloWorld)
//sampleEnd
}
```
