## 🧬 泛型

#### 🔍 Kotlin 中的类可以有类型参数，与 Java 类似：

```kotlin
class Person<T>(t: T) {
    var value = t
}
```

#### 🏗️ 创建这样类的实例只需提供类型参数即可：

```kotlin
val person: Person<Int> = Person<Int>(1)
```

#### 🧐 但是如果类型参数可以推断出来，例如从构造函数的参数或者从其他途径，就可以省略类型参数：

```kotlin
val person = Person(1) // 1 具有类型 Int，所以编译器推算出它是 Box<Int>
