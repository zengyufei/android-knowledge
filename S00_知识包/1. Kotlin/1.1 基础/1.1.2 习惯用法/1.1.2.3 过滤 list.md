### 🔍 过滤 list
```kotlin
val positives = list.filter { x -> x > 0 }
```
#### ✂️ 或者可以更短:

```kotlin
val positives = list.filter { it > 0 }
```

🖨️ 实例：

```kotlin
val list = listOf(1, 2, 3, 4, 5, 6, 7, 8, 9)
val positives = list.filter { x -> x > 5 }
// 或 val positives = list.filter { it > 5 }
fun main(){
    println(positives)
}

// [6, 7, 8, 9]
```
