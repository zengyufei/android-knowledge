### 🔧 配置对象的属性（apply）

```kotlin
class Person {
    var name: String = ""
    var age: Int = 0
}

fun main() {
    val person = Person().apply {
        name = "Alice" // 直接访问属性
        age = 30
    }

    println("Name: ${person.name}, Age: ${person.age}") // 输出: Name: Alice, Age: 30
}
```

> 🛠️ <code>apply</code> 函数是一个扩展函数，用于在对象的上下文中执行代码块，方便地初始化和配置对象，同时返回该对象本身。
